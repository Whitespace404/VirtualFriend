# VirtualFriend
A virtual friend made using Cognimates and Uclassify. 

# Setup
1) Download a zip file of this repository.
2) Unzip the downloaded zip file.
3) Go to https://codelab.cognimates.me/.
4) Click on File >> Load from your Computer.
4) Open the Folder named "VirtualFriend". 
5) Load the file named "Virtual Friend v2.0"
6) Click the green flag to start the virtual friend.
7) Hold the spacebar to unmute yourself to talk. Cognimates will ask for your mic permission.

# Questions you can ask:

1) What is your name?
2) What is the time?
3) Where do you live?
4) What is your hobby?
